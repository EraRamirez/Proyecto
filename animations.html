Procedimiento para resolver el producto de dos matrices por Strassen

Proceso simplificado con fines prácticos por lo que puede ser posible que se necesite repetir el proceso más de una vez para obtener el resultado

Explicación

Paso 1

Segmentar nuestra matriz 'A' o 'B' de tamaño 2n en 4 submatrices, como lo siguiente...

Paso 2

Obtener las ecuaciones 'P', que son las que nos ayudarán a resolver el producto.

Paso 3

Para el paso 3, lo que haremos ahora es usar las 7 P's que obtuvimos en el paso anterior y las juntamos en 4 ecuaciones para obtener la matriz resultante.

M1, 1 = P5 + P4 - P2 + P6

M1, 2 = P1 + P2

M2, 1 = P3 + P4

M2, 2 = P1 + P5 - P3 - P7

Paso 4

En el paso 4 lo único que tenemos que hacer es juntar las M's para obtener la matriz resultante

Representación en código

let a, b, c, d, e, f, g, h, halfA=A.length/2, halfB=B.length/2;
a = getSubMatrix(A, 0, halfA, 0, halfA);
b = getSubMatrix(A, halfA, 0, halfA);
c = getSubMatrix(A, 0,halfA, halfA, A.length);
d = getSubMatrix(A, halfA, A.length, halfA, A.length);
e = getSubMatrix(B, 0, halfB, 0, halfB);
f = getSubMatrix(B, halfB, B.length, 0, halfB);
g = getSubMatrix(B, 0, halfB, halfB, B.length);
h = getSubMatrix(B, halfB, B.length, halfB, B.length);
let p1 = strassen(a, subtractionMatrix(f, h)),
p2 = strassen(additionMatrix(a, b), h),
p3 = strassen(additionMatrix(c, d), e),
p4 = strassen(d, subtractionMatrix(g, e)),
p5 = strassen(additionMatrix(a, d), additionMatrix(e, h)),
p6 = strassen(subtractionMatrix(b, d), additionMatrix(g, h)),
p7 = strassen(subtractionMatrix(a, c), additionMatrix(e, f)),
f1 = additionMatrix(p5, p4),
f2 = subtractionMatrix(f1, p2),
c11 = additionMatrix(f2, p6),
c12 = additionMatrix(p1, p2),
c21 = additionMatrix(p3, p4);
f1 = additionMatrix(p1, p5);
f2 = subtractionMatrix(f1, p3);
let c22 = subtractionMatrix(f2, p7);
let result = concatMatrix(c11, c12, c21, c22);
return result;

Producto de matrices usando el método tradicional

Cuando hacemos el producto de matrices por el método tradicional, es el más simple que podemos hacer.

Sin embargo puede llegar a ser más tardado que otros métodos.

En éste, tenemos que tomar una Fila de la matriz A y multiplicar el Elemento Ai, j y multiplicarlo por el Elemento Bj, k de la Matriz B

Y después sumaremos así por cada fila; de tal forma que la ecuación general es: ΣAi, j * Bj, k
